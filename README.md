To-Do List Application (Python)

📌 Description

This is a simple To-Do List application built using Python. It helps users manage their daily tasks efficiently by allowing them to add, view, delete, and mark tasks as completed.

🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Simple and user-friendly interface

🛠️ Technologies Used

- Python (Core Python concepts)
- File Handling (for saving tasks)

▶️ How to Run

1. Install Python on your system
2. Clone this repository
3. Run the Python file:
   python todo.py

📂 Project Structure

- "todo.py" → Main program file
- "tasks.txt" → Stores tasks (if file handling used)

🎯 Purpose

This project is created for learning basic Python concepts like loops, functions, and file handling.
